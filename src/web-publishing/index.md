# Webサイトを公開する

作成したWebサイトを無料で公開する方法を紹介します。

## GitHubを使う

GitHub <https://github.com> を使うことでより本格的にWebサイトを公開できます (このガイドもそうです)。

GitHubを使うにはアカウントの作成が必要です。まずGitHubの無料アカウントを作成しましょう。

Webページを公開する流れ:

1. [GitHubでのアカウントの作成](https://docs.github.com/ja/get-started/quickstart/creating-an-account-on-github)
2. [GitHub Pagesサイトの作成](https://docs.github.com/ja/pages/getting-started-with-github-pages/creating-a-github-pages-site)

## Cloudflare Pagesを使う

Cloudflareが提供する [Cloudflare Pages](https://www.cloudflare.com/drop/) のドラッグ&ドロップ機能を使うと、アカウントを作らなくてもフォルダをアップロードするだけですぐにWebサイトを公開できます。

Webページを公開する流れ:

1. <https://www.cloudflare.com/drop/> にアクセスする
2. 公開したいHTMLファイルなどが入ったフォルダを、ページ上にドラッグ&ドロップする
3. アップロードが完了すると公開用のURLが発行される

公式ドキュメント: [Direct Upload (Cloudflare Pages)](https://developers.cloudflare.com/pages/get-started/direct-upload/)

サイトを継続して管理・更新したい場合は無料のCloudflareアカウント作成が必要です。

## Netlify Dropを使う

Netlifyが提供する [Netlify Drop](https://app.netlify.com/drop) も、Cloudflare Pagesと同様にフォルダをドラッグ&ドロップするだけでWebサイトを公開できるサービスです。

Webページを公開する流れ:

1. <https://app.netlify.com/drop> にアクセスする
2. 公開したいHTMLファイルなどが入ったフォルダを、ページ上にドラッグ&ドロップする
3. アップロードが完了すると公開用のURLが発行される

公式ドキュメント: [Manual deploys (Netlify Docs)](https://docs.netlify.com/deploy/deploys/manual-deploys/)

サイトを継続して管理・更新したい場合は無料のNetlifyアカウント作成が必要です。

自分に合った方法でWebサイトを公開してみましょう!
